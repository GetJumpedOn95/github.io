ISP

The concept of my ISP is a text based adventure that you progress through by making 2 choices.One of the choices will result in failure while the other choice will allow you to progress.We wanted to use the text based medium to create a creative and enriching story that will compel the player to keep playing and progress to see what will happen next.One hardship we had to overcome was the game board not fitting the screen of the computer because it was based on the size of the moniter.To fix this we had to create a code that instead fit the game board to the size of the screen so we could present the game board in class.One thing that did go as planned was the squares that showed up as you progressed to show how far you got.We even got them to stay so that it would show you how far you got the previous attempt.

Important Classes in Project

Background.swift - In charge of drawing the game board

InteractionLayer.swift - In Charge of moving the player around the board and keeping track of which question the player was on

Player.swift - Worked with the interactionLayer to move the player around the board and actually drew the player's spot on the board

QuestionBoard.swift - This class was in charge of popping up questions and the users actions
