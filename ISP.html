ISP

The concept of my ISP is a text based adventure that you progress through by making 2 choices.One of the choices will result in failure while the other choice will allow you to progress.We wanted to use the text based medium to create a creative and enriching story that will compell the player to keep playing and progress to see what will happen next.One hardship we had to overcome was the game board not fitting the screen of the computer because it was based on the size of the moniter.To fix this we had to create a code that instead fit the game board to the size of the screen so we could present the game board in class.One thing that did go as planed was the squares that showed up as you progressed to show how far you got.We even got them to stay so that it would show you how far you got the previous attempt.
